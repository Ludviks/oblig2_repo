Git/Github

GitHub er en webbasert hosting service for version control. Det er en samarbeidsplattform for software utviklere. Microsoft er den største investoren til GitHub, i og med at de kjøpte dem for 7.5 milliarder dollar i Juni 2018. GitHub er levert som en software som en tjeneste, eller en SaaS (software as a service). Det ble startet i 2008 i Git som da er et styringssytem med åpen kildekode, og er laget av Linus Torvald for å lage software builds fortere

Media typer

Media typer er en CSS teknikk som ble introdusert i CSS3, den går ut på å bruke @media taggen til å bruke en blokk med CSS kode bare hvis ett tilfeller er true.

@media only screen and (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

Dette eksempelet går ut på at hvis skjermen er 600px eller mindre så skal bakgrunnen være lyseblå

Css layouts

Layouts i css har utviklet seg mye i det siste, disse nye metodene er med på å skape en bedre struktur i nettsidene. En av disse metodene er Grid, det er en struktur med rader og kolonner som utgjør celler. En annen metode er den nye flexbox metoden som erstatter grid. Det fjerner i mange tilfeller behovet for float og avanserte layouts. Den lager fleksible bokser, og er støtter i aller moderne nettlesere, og er veldig lett å bruke når man først forstår det.
